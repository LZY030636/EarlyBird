# EarlyBird
基于机器视觉的脚本框架

## 关于本项目
EarlyBird是一款通用视觉脚本框架，目前仍处于Beta版本。  
通过该框架，你可以自定义自己的机器视觉脚本完成机械化的无聊操作。  
同时，框架内置一套图书馆抢座脚本策略，图书馆抢座也是本框架目前主要的开发方向。
